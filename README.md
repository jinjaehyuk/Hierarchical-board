## 구현기능 및 필수기술
- 구현 기능
    - 계층형 댓글 게시판
        - 99 Depth까지 가능하도록
        - 댓글이 많아도 속도 저하를 최소화 할 수 있는 방법 적용
    - 기본적인 스크립트 공격 방어
    - 이미지 등록
    - 로그인/회원가입 등 기본적인 회원 기능
- 필수 기술
    - Java 11
    - SpringBoot
    - Git & Gitlab
    - Gradle
    - MySQL
    - Javascript
